了解了，只介绍 C++11 中与线程相关的知识点，下面详细介绍 C++11 线程库的关键组件和功能：

1. std::thread

std::thread 类用于创建和管理线程。每个 std::thread 对象表示一个线程，能够执行任何可调用的对象。

创建线程

  • 构造函数：创建线程并执行指定的函数或可调用对象。

    #include <iostream>
#include <thread>

void print_message(const std::string& message) {
    std::cout << message << std::endl;
}

int main() {
    std::thread t(print_message, "Hello from thread!"); // 创建线程并传递参数
    t.join(); // 等待线程完成
    return 0;
}

  • **join() 和 detach()**：

    • **join()**：等待线程执行完成。主线程会阻塞，直到 join() 的线程完成。
    • **detach()**：将线程分离，让它在后台继续运行，不会阻塞主线程。如果线程对象超出作用域，分离线程的资源会被自动释放，但主线程不会等待它完成。
    std::thread t(print_message, "Hello from detached thread!");
t.detach(); // 让线程在后台运行

2. std::this_thread

std::this_thread 提供了与当前线程相关的功能。

常用功能

  • **std::this_thread::sleep_for()**：让当前线程休眠指定的时间段。

    #include <thread>
#include <chrono>

int main() {
    std::this_thread::sleep_for(std::chrono::seconds(2)); // 休眠 2 秒
    return 0;
}

  • **std::this_thread::sleep_until()**：让当前线程休眠直到指定的时间点。

    #include <thread>
#include <chrono>

int main() {
    auto wake_up_time = std::chrono::steady_clock::now() + std::chrono::seconds(2);
    std::this_thread::sleep_until(wake_up_time); // 休眠直到指定时间点
    return 0;
}

  • **std::this_thread::get_id()**：获取当前线程的 ID。

    #include <iostream>
#include <thread>

int main() {
    std::cout << "Current thread ID: " << std::this_thread::get_id() << std::endl;
    return 0;
}

3. std::future 和 std::promise

std::future 和 std::promise 用于异步操作的结果传递和同步。

std::promise 和 std::future

  • **std::promise**：用于设置一个值或异常，供其他线程通过 std::future 获取。

    #include <iostream>
#include <thread>
#include <future>

void produce(std::promise<int>&& prom) {
    std::this_thread::sleep_for(std::chrono::seconds(1));
    prom.set_value(42); // 设置结果值
}

int main() {
    std::promise<int> prom;
    std::future<int> fut = prom.get_future(); // 获取 future 对象
    std::thread t(produce, std::move(prom)); // 将 promise 移动到线程
    std::cout << "Waiting for result..." << std::endl;
    std::cout << "Result: " << fut.get() << std::endl; // 获取结果
    t.join();
    return 0;
}

  • **std::future**：用于获取异步操作的结果。可以通过 get() 方法获取结果，或者使用 wait() 和 wait_for() 等方法等待结果的到来。

    #include <iostream>
#include <thread>
#include <future>

int compute_value() {
    std::this_thread::sleep_for(std::chrono::seconds(1));
    return 42;
}

int main() {
    std::future<int> fut = std::async(std::launch::async, compute_value); // 异步计算
    std::cout << "Waiting for result..." << std::endl;
    std::cout << "Result: " << fut.get() << std::endl; // 获取结果
    return 0;
}

4. std::async

std::async 用于启动一个异步任务并返回 std::future 对象，可以用于获取异步操作的结果。

使用 std::async

  • **std::launch::async**：强制异步执行。

    #include <iostream>
#include <thread>
#include <future>

int compute_value() {
    std::this_thread::sleep_for(std::chrono::seconds(1));
    return 42;
}

int main() {
    std::future<int> fut = std::async(std::launch::async, compute_value); // 异步计算
    std::cout << "Waiting for result..." << std::endl;
    std::cout << "Result: " << fut.get() << std::endl; // 获取结果
    return 0;
}

  • **std::launch::deferred**：推迟执行，直到 get() 或 wait() 被调用。

    #include <iostream>
#include <thread>
#include <future>

int compute_value() {
    std::cout << "Computing..." << std::endl;
    return 42;
}

int main() {
    std::future<int> fut = std::async(std::launch::deferred, compute_value); // 推迟执行
    std::cout << "Waiting for result..." << std::endl;
    std::cout << "Result: " << fut.get() << std::endl; // 执行并获取结果
    return 0;
}

5. std::atomic

std::atomic 提供原子操作，用于处理共享数据时避免数据竞争。适用于基本数据类型，如 int、bool 等。

使用 std::atomic

#include <iostream>
#include <thread>
#include <atomic>

std::atomic<int> counter(0);

void increment() {
    for (int i = 0; i < 1000; ++i) {
        ++counter; // 原子操作，避免数据竞争
    }
}

int main() {
    std::thread t1(increment);
    std::thread t2(increment);
    t1.join();
    t2.join();
    std::cout << "Counter: " << counter << std::endl;
    return 0;
}

原子操作的常用方法

  • **store()**：设置值。
  • **load()**：获取值。
  • **fetch_add()、fetch_sub()**：原子性加法和减法。
  • **compare_exchange_weak()、compare_exchange_strong()**：原子比较和交换操作。

总结

C++11 的线程库引入了多个功能强大的组件来支持多线程编程：

  • **std::thread**：创建和管理线程。
  • **std::this_thread**：提供与当前线程相关的功能，如休眠和获取线程 ID。
  • **std::future 和 std::promise**：用于异步操作的结果传递和同步。
  • **std::async**：启动异步任务并返回 std::future 对象。
  • **std::atomic**：提供原子操作，处理多线程中的基本数据类型。

这些工具和类使得 C++11 在处理并发编程时更加强大和灵活。
